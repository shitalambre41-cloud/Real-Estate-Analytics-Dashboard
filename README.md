# Real-Estate-Analytics-Dashboard
This Power BI dashboard provides an interactive and comprehensive analysis of real estate property performance. It tracks key metrics such as Total Properties, Sold, For Sale, and Pending listings, with KPI cards and dynamic progress indicators.The dashboard visualizes price trends, property status distribution, agent-wise listings, and days-on-market breakdown to help understand market behavior and sales efficiency.

It also includes a detailed property-level table displaying MLS ID, location, area, lot size, price, and status, allowing users to drill down into individual property details. Additional charts showcase market status by property type and time-based performance trends, supporting data-driven decision-making for real estate agencies, buyers, and analysts.

Key Highlights:
KPI cards for major property status categories
Agent performance panel
Days-on-market vs. price trend analysis
Property type market distribution
Detailed property listing table for granular insights
Clean, modern UI with a real-estate themed design
